## Tech Stack
- PostgreSQL
- Ruby on Rails
- ReactJS
- Redux

## Setup

```sh
npm install
```

## Running Development Server

From this directory:

```sh
npm start
```

Opens at http://localhost:8080 and proxies `/api` to the local Rails server at http://localhost:3000.
